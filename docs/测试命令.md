# 流式接口并发测试命令

## 基础测试

```bash
# 使用默认配置
uv run python src/stream_test_enhanced.py -c configs/config.json

# 使用命令行参数
uv run python src/stream_test_enhanced.py -u https://api.example.com/stream -n 50 -p 10
```

## Dify API 测试

```bash
# 使用固定查询内容
uv run python src/stream_test_enhanced.py -c configs/config_dify.json

# 使用CSV数据源（推荐）
uv run python src/stream_test_enhanced.py -c configs/config_dify_with_csv.json
```

**注意**：使用前需要将配置文件中的 `{api_key}` 替换为实际的 API Key。

## 其他配置

```bash
# 使用CSV数据源配置
uv run python src/stream_test_enhanced.py -c configs/config_csv_example.json
```

## 参数说明

- `-c, --config`: 配置文件路径
- `-u, --url`: 接口URL
- `-n, --requests`: 总请求数（默认100）
- `-p, --concurrency`: 并发数（默认10）

## 详细文档

- Dify配置说明：查看 `DIFY_CONFIG.md`
- CSV数据源：查看 `README.md` 中的 CSV数据源配置部分
